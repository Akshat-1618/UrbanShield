# UrbanShield

Smart City Emergency Command Platform

## Features

- Emergency Reporting
- Real-Time Dispatch
- Incident Tracking
- Route Optimization
- Resource Allocation
- Heatmap Analytics
- Road Blockage Simulation

## Tech Stack

Frontend:
- React
- Tailwind
- Leaflet

Backend:
- Node.js
- Express

Database:
- MongoDB

Realtime:
- Socket.IO

Algorithms:
- BFS
- Floyd Warshall
- Priority Queue
